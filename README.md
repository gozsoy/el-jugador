# el-jugador
A self-sufficient CLI game for practising Spanish verb conjugations.

![gameplay](demo.gif)

## Features
- Practice across 600+ verbs and 10 tenses
- Save new verbs to your personal list
- Choose to quiz all verbs or only your saved verbs
- Ask all 6 subjects or 1 random subject per verb
- Type `hint` anytime to reveal the answer

## Mac / Linux

```bash
make run
```

## Windows

1. Install [Git for Windows](https://gitforwindows.org)
2. Double-click `run.bat`
